# projet_3_vols_avions_nantes
Bienvenue sur notre projet !

Nous sommes un groupe de 3 supers data analystes en formation et nous avons crée cette application Streamlit.
Comme nous sommes proches de l'aéroport de Nantes, notre but premier était de pouvoir identifier les avions qui passent au dessus de nos têtes.
Puis finalement nous avons décidé de pousser plus loin en fournissant les statistiques intéressantes sur l'espace aérien français.
Nous avons aussi en proposé une recherche de vol en temps réel, avec du machine learning nous sélectionnons le meilleur vol pour vous.
